# Тесты для библиотеки decpy

Проект содержит набор автоматических тестов для библиотеки [decpy], реализующей ленивые вычисления в Python.

## О проекте

Библиотека `decpy` позволяет описывать запросы к данным (фильтры, соединения, агрегации) в «ленивом» стиле — выражения не вычисляются сразу, а откладываются до момента, когда появятся реальные данные. В рамках дипломной работы я написал тесты, которые покрывают **все активные примеры** из отладочных скриптов, предоставленных автором библиотеки.

Каждый тест соответствует одному выводу (`print`) из соответствующего отладочного файла и проверяет корректность работы библиотеки.

## Что проверяют тесты

- **test_01.py** – ленивые формулы, индексы, срезы, фильтры, декартово произведение, автоматические множества классов (`Otladka_lenivykh.py`).
- **test_02.py** – проекции и вычисления над множествами, векторами, вложенными кортежами (`Otladka_proektsiy_i_vychisleniy.py`).
- **test_03.py** – работа с таблицами продаж, группировка, агрегации (`Baza_dannykh_Prodazhi.py`).
- **test_04.py** – исчисление на доменах, соединения, полёты самолётов (`Otladka_ischislenia_na_domenakh.py`).
- **test_05.py** – логическое программирование: родственные связи (`Otladka_logicheskogo_programmirovania.py`).
- **test_06.py** – отладка таблиц-предикатов (`otladka_tablits-predikatov.py`).
- **test_07.py** – родословное древо через ООП (`Otladka_rodoslovnogo_dreva_OOP.py`).
- **test_08.py** – родословное древо через предикаты (`Rodoslovnoe_drevo_Predikaty.py`).

Каждый файл содержит ровно столько тестов, сколько активных `print` в соответствующем отладочном скрипте (без заголовков и закомментированных блоков).

## Как запустить

1. Установите зависимости:
   ```bash
   pip install pytest decpy
2. Клонируйте репозиторий:
   git clone https://github.com/ваш-логин/decpy-tests.git
   cd decpy-tests
3. Запустите все тесты:
   pytest -v
Все тесты проходят на версии decpy 2.0.0.
Структура проекта:

test_01.py … test_08.py – тестовые файлы
Otladka_*.py, Baza_dannykh_Prodazhi.py, Rodoslovnoe_drevo_Predikaty.py – отладочные скрипты, по которым писались тесты (если вы решите их тоже загрузить)
README.md – это описание
